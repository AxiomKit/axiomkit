# Gate.io Wallet (Concept)

Gate.io wallet support in AxiomKit is **planned** as an upcoming wallet provider.  
The goal is to let you plug a Gate.io wallet into your agent **the same way you would plug in any other wallet provider**, using the standard AxiomKit provider pattern.


## Target Features

These features are **design goals** and may evolve as the provider is implemented:

- **Balance & portfolio access (planned)**: Read spot/futures balances and key portfolio metrics.
- **Trading & orders (planned)**: Place and manage orders through agent tools with safety checks.
- **Multi-wallet pattern (planned)**: Use Gate.io alongside other wallets/providers in a single agent.
- **Configurable permissions (planned)**: Limit what the agent can do (read-only, paper trading, etc.).

## Status

Gate.io wallet support is **not yet released**.  
This page describes the **intended integration pattern**, so you can design agents now and swap in the official provider once it becomes available.
