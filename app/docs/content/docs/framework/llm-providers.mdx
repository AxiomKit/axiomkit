# LLM Providers

## Overview

Axiomkit is designed to be model-agnostic, allowing you to use any language model provider that supports the `@ai-sdk` interface. This gives you the flexibility to choose the best model for your specific use case, whether it's for cost, performance, or feature requirements.

## Supported Providers

Axiomkit supports all providers that work with the `@ai-sdk` library:

### 🤖 OpenAI

OpenAI's models including GPT-4, GPT-3.5, and GPT-4 Turbo.

```typescript
import { createAgent } from "@axiomkit/core";
import { openai } from "@ai-sdk/openai";

const agent = createAgent({
  model: openai("gpt-4"),
  modelSettings: {
    maxTokens: 1000,
    temperature: 0.7,
  },
});

// Or use GPT-4 Turbo
const turboAgent = createAgent({
  model: openai("gpt-4-turbo-preview"),
  modelSettings: {
    maxTokens: 4000,
    temperature: 0.5,
  },
});
```

**Best for:**
- High-quality reasoning and analysis
- Complex problem-solving
- Creative writing
- Code generation

### 🚀 Groq

Fast inference with models like DeepSeek, Llama, and Mixtral.

```typescript
import { createAgent } from "@axiomkit/core";

const agent = createAgent({
  model: groq("deepseek-r1-distill-llama-70b"),

    maxTokens: 1000,
    temperature: 0.7,
  },
});

// Use different models
const llamaAgent = createAgent({
  model: groq("llama-3-8b-8192"),
});

const mixtralAgent = createAgent({
  model: groq("mixtral-8x7b-32768"),
});
```

**Best for:**
- Fast response times
- Cost-effective inference
- Real-time applications
- High-throughput scenarios

### 🔧 Custom Providers

You can also use custom providers or create your own.

```typescript
import { createAgent } from "@axiomkit/core";
import { custom } from "@ai-sdk/custom";

const agent = createAgent({
  model: custom({
    id: "my-custom-model",
    generate: async (prompt) => {
      // Your custom generation logic
      const response = await myCustomAPI.generate(prompt);
      return response.text;
    },
  }),
});
```

## Model Configuration

### Basic Settings

```typescript
const agent = createAgent({
  model: groq("deepseek-r1-distill-llama-70b"),
  modelSettings: {
    maxTokens: 1000,        // Maximum tokens to generate
    temperature: 0.7,       // Creativity (0.0 = deterministic, 1.0 = very creative)
    topP: 0.9,             // Nucleus sampling parameter
    frequencyPenalty: 0.0,  // Reduce repetition
    presencePenalty: 0.0,   // Encourage new topics
  },
});
```

### Advanced Settings

```typescript
const agent = createAgent({
  model: openai("gpt-4"),
  modelSettings: {
    maxTokens: 2000,
    temperature: 0.3,
    topP: 0.95,
    frequencyPenalty: 0.1,
    presencePenalty: 0.1,
    stop: ["\n\n", "Human:", "Assistant:"], // Stop generation at these tokens
    logitBias: {
      "positive": 1.0,      // Bias towards positive words
      "negative": -1.0,     // Bias away from negative words
    },
  },
});
```

## Model Selection Guide

### For Different Use Cases

| Use Case | Recommended Model | Reason |
|----------|------------------|---------|
| **Fast Responses** | Groq (DeepSeek/Llama) | Low latency, cost-effective |
| **Complex Reasoning** | OpenAI GPT-4, Claude Opus | Advanced reasoning capabilities |
| **Creative Writing** | GPT-4, Claude Sonnet | Strong creative abilities |
| **Code Generation** | GPT-4, CodeLlama | Specialized for code |
| **Privacy-Sensitive** | Ollama (local) | No data leaves your system |
| **Cost-Effective** | Groq, Gemini Flash | Lower cost per token |
| **Multimodal** | Gemini, GPT-4V | Image and text understanding |

### Multiple Providers

You can use multiple providers in the same application:

```typescript
import { createAgent } from "@axiomkit/core";
import { openai } from "@ai-sdk/openai";
import { groq } from "@ai-sdk/groq";
import { anthropic } from "@ai-sdk/anthropic";

// Different agents for different purposes
const fastAgent = createAgent({ model: groq("deepseek-r1-distill-llama-70b") });
const qualityAgent = createAgent({ model: openai("gpt-4") });
const reasoningAgent = createAgent({ model: anthropic("claude-3-sonnet-20240229") });

// Use the appropriate agent for each task
async function handleRequest(task: string, priority: "speed" | "quality" | "reasoning") {
  switch (priority) {
    case "speed":
      return await fastAgent.run({ args: { message: task } });
    case "quality":
      return await qualityAgent.run({ args: { message: task } });
    case "reasoning":
      return await reasoningAgent.run({ args: { message: task } });
  }
}
```

const costOptimizedAgent = createAgent({

    maxTokens: 500,        // Limit response length

### Provider Switching

```typescript
class AdaptiveAgent {
  private fastModel = groq("deepseek-r1-distill-llama-70b");
  private qualityModel = openai("gpt-4");
  
  async run(input: string, budget: "low" | "high") {
    const model = budget === "low" ? this.fastModel : this.qualityModel;
    const agent = createAgent({ model });
    
    return await agent.run({ args: { message: input } });
  }
}
```

## Error Handling

### Provider-Specific Errors

```typescript
try {
  const result = await agent.run({
    context: { type: "chat" },
    args: { message: "Hello" }
  });
  return result.output;
} catch (error) {
  if (error.code === "rate_limit_exceeded") {
    // Handle rate limiting
    console.log("Rate limit exceeded, retrying later...");
    return "I'm busy right now, please try again in a moment.";
  } else if (error.code === "insufficient_quota") {
    // Handle quota issues
    console.log("API quota exceeded");
    return "Service temporarily unavailable.";
  } else {
    // Handle other errors
    console.error("Model error:", error);
    return "I encountered an error. Please try again.";
  }
}
```

### Fallback Strategies

```typescript
class ResilientAgent {
  private models = [
    groq("deepseek-r1-distill-llama-70b"),
    openai("gpt-3.5-turbo"),
    anthropic("claude-3-haiku-20240307")
  ];
  
  async run(input: string) {
    for (const model of this.models) {
      try {
        const agent = createAgent({ model });
        const result = await agent.run({ args: { message: input } });
        return result.output;
      } catch (error) {
        console.log(`Model ${model} failed, trying next...`);
        continue;
      }
    }
    throw new Error("All models failed");
  }
}
```

## Best Practices

### 1. Model Selection

- **Start with Groq** for fast, cost-effective development
- **Use GPT-4 or Claude** for production quality applications
- **Consider Ollama** for privacy-sensitive use cases
- **Test multiple models** to find the best fit for your use case

### 2. Configuration

- **Set appropriate maxTokens** to control costs
- **Use lower temperature** for consistent, factual responses
- **Implement retry logic** for reliability
- **Monitor usage** to optimize costs

### 3. Error Handling

- **Handle rate limits** gracefully
- **Implement fallback models** for reliability
- **Log errors** for debugging
- **Provide user-friendly error messages**

### 4. Performance

- **Cache responses** when appropriate
- **Use streaming** for long responses
- **Batch requests** when possible
- **Monitor latency** and optimize accordingly

## Testing Different Models

```typescript
async function testModels(prompt: string) {
  const models = [
    { name: "Groq DeepSeek", model: groq("deepseek-r1-distill-llama-70b") },
    { name: "OpenAI GPT-4", model: openai("gpt-4") },
    { name: "Claude Sonnet", model: anthropic("claude-3-sonnet-20240229") },
  ];
  
  const results = [];
  
  for (const { name, model } of models) {
    const start = Date.now();
    try {
      const agent = createAgent({ model });
      const result = await agent.run({ args: { message: prompt } });
      const duration = Date.now() - start;
      
      results.push({
        model: name,
        response: result.output,
        duration,
        success: true
      });
    } catch (error) {
      results.push({
        model: name,
        error: error.message,
        duration: Date.now() - start,
        success: false
      });
    }
  }
  
  return results;
}

// Test different models
const testResults = await testModels("Explain quantum computing in simple terms");
console.table(testResults);
```
