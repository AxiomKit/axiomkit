## Overview

**Axiomkit** is a framework for building autonomous AI agents capable of interacting with real-world systems. At its core, each agent follows a continuous execution loop:

1. **Analyze** inputs received from users or external systems.
2. **Reason** about those inputs using a large language model (LLM)
3. **Decide** what to do next: generate an output or trigger an action.
4. **Reflect** on the outcome, feeding results back into the agent’s context
