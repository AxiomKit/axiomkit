# First AI Agent

Learn how to build your first AI agent with Axiomkit. This guide will walk you through creating a simple calculator bot that can perform mathematical operations.

## What We'll Build

We'll create a calculator agent that can:
- Perform basic mathematical operations
- Show step-by-step work for complex problems
- Remember previous calculations
- Handle errors gracefully

## Project Setup

### 1. Create Project Structure

```bash
mkdir calculator-agent
cd calculator-agent
npm init -y
```

### 2. Install Dependencies

```bash
npm install @axiomkit/core @axiomkit/cli
npm install @ai-sdk/groq
npm install -D typescript @types/node
```

### 3. Create TypeScript Configuration

```bash
npx tsc --init
```

Update `tsconfig.json`:

```json
{
  "compilerOptions": {
    "target": "ES2020",
    "module": "commonjs",
    "outDir": "./dist",
    "rootDir": "./src",
    "strict": true,
    "esModuleInterop": true,
    "skipLibCheck": true,
    "forceConsistentCasingInFileNames": true
  },
  "include": ["src/**/*"],
  "exclude": ["node_modules", "dist"]
}
```

## Building the Calculator Agent

### 1. Create the Main Agent File

Create `src/calculator-agent.ts`:

```typescript
import { createAgent } from "@axiomkit/core";
import { createCliExtension } from "@axiomkit/cli";
import { groq } from "@ai-sdk/groq";

// Define the calculator extension
const calculatorExtension = createCliExtension({
  name: "calculator",
  instructions: [
    "You are a mathematical calculator assistant.",
    "When users provide mathematical expressions, calculate the result.",
    "Show your work step by step for complex problems.",
    "Support basic operations: +, -, *, /, ^, sqrt, etc.",
    "Always verify your calculations and explain your reasoning.",
    "If you encounter an error, explain what went wrong.",
  ],
  examples: [
    {
      input: "What is 15 + 27?",
      output: "15 + 27 = 42\n\nThe calculation is straightforward addition.",
    },
    {
      input: "Calculate 8 * 7 + 3",
      output: "8 * 7 + 3\n= 56 + 3\n= 59\n\nI followed the order of operations: multiplication first, then addition.",
    },
    {
      input: "What is the square root of 144?",
      output: "√144 = 12\n\n12 * 12 = 144, so the square root of 144 is 12.",
    },
  ],
});

// Create the agent
const agent = createAgent({
  model: groq("gemma2-9b-it"),
  modelSettings: {
    maxTokens: 600,
    temperature: 0.1, // Low temperature for consistent calculations
  },
  extensions: [calculatorExtension],
  memory: {
    enabled: true,
    maxEpisodes: 10, // Remember last 10 interactions
  },
});

async function main() {
  try {
    // Start the agent
    await agent.start();
    
    console.log("🧮 Calculator Agent started!");
    console.log("Type mathematical expressions or questions.");
    console.log("Examples:");
    console.log("  - What is 15 + 27?");
    console.log("  - Calculate 8 * 7 + 3");
    console.log("  - What is the square root of 144?");
    console.log("Type 'exit' to quit\n");
    
    // Simple CLI interface
    process.stdin.on('data', async (data) => {
      const input = data.toString().trim();
      
      if (input === 'exit') {
        console.log("👋 Goodbye!");
        await agent.stop();
        process.exit(0);
      }
      
      if (!input) return;
      
      try {
        console.log("🤔 Processing...");
        
        const result = await agent.run({
          context: {
            type: "calculator",
            instructions: "You are a helpful calculator. Provide clear, step-by-step solutions.",
          },
          args: { 
            message: input,
            timestamp: new Date().toISOString(),
          },
        });
        
        console.log("🧮 Result:", result.output);
        console.log("---");
        
      } catch (error) {
        console.error("❌ Error:", error.message);
        console.log("---");
      }
    });
    
  } catch (error) {
    console.error("Failed to start agent:", error);
    process.exit(1);
  }
}

// Handle graceful shutdown
process.on('SIGINT', async () => {
  console.log("\n👋 Shutting down...");
  await agent.stop();
  process.exit(0);
});

main().catch(console.error);
```

## Adding Advanced Features

### 1. Custom Actions

Let's add a custom action for more complex calculations:

```typescript
import { z } from "zod";

const advancedMathAction = {
  name: "advanced_math",
  description: "Perform advanced mathematical operations",
  schema: z.object({
    operation: z.enum(["factorial", "fibonacci", "prime_check"]),
    number: z.number(),
  }),
  handler: async (args, ctx, agent) => {
    switch (args.operation) {
      case "factorial":
        const factorial = (n: number): number => n <= 1 ? 1 : n * factorial(n - 1);
        return { result: factorial(args.number) };
        
      case "fibonacci":
        const fibonacci = (n: number): number => {
          if (n <= 1) return n;
          return fibonacci(n - 1) + fibonacci(n - 2);
        };
        return { result: fibonacci(args.number) };
        
      case "prime_check":
        const isPrime = (n: number): boolean => {
          if (n < 2) return false;
          for (let i = 2; i <= Math.sqrt(n); i++) {
            if (n % i === 0) return false;
          }
          return true;
        };
        return { result: isPrime(args.number) };
        
      default:
        throw new Error(`Unknown operation: ${args.operation}`);
    }
  },
};

// Add to agent configuration
const agent = createAgent({
  model: groq("gemma2-9b-it"),
  modelSettings: {
    maxTokens: 600,
    temperature: 0.1,
  },
  extensions: [calculatorExtension],
  actions: [advancedMathAction],
  memory: {
    enabled: true,
    maxEpisodes: 10,
  },
});
```

### 2. Error Handling

Add better error handling:

```typescript
// Enhanced error handling
process.stdin.on('data', async (data) => {
  const input = data.toString().trim();
  
  if (input === 'exit') {
    console.log("👋 Goodbye!");
    await agent.stop();
    process.exit(0);
  }
  
  if (!input) return;
  
  try {
    console.log("🤔 Processing...");
    
    const result = await agent.run({
      context: {
        type: "calculator",
        instructions: "You are a helpful calculator. Provide clear, step-by-step solutions.",
      },
      args: { 
        message: input,
        timestamp: new Date().toISOString(),
      },
      handlers: {
        onError: (error) => {
          console.error("❌ Agent Error:", error.message);
        },
        onLogStream: (log, done) => {
          if (log.ref === "thought" && !done) {
            console.log("💭 Thinking:", log.content);
          }
        },
      },
    });
    
    console.log("🧮 Result:", result.output);
    console.log("---");
    
  } catch (error) {
    if (error.message.includes("rate limit")) {
      console.error("⚠️ Rate limit exceeded. Please wait a moment.");
    } else if (error.message.includes("invalid model")) {
      console.error("⚠️ Model not available. Check your configuration.");
    } else {
      console.error("❌ Error:", error.message);
    }
    console.log("---");
  }
});
```

## Best Practices

### 1. Model Selection

- Use lower temperature (0.1-0.3) for mathematical tasks
- Higher temperature (0.7-0.9) for creative tasks
- Adjust maxTokens based on expected response length

### 2. Memory Management

```typescript
const agent = createAgent({
  // ... other config
  memory: {
    enabled: true,
    maxEpisodes: 10,
    pruningStrategy: "lru", // Least recently used
    compressionEnabled: true,
  },
});
```

### 3. Context Design

```typescript
const calculatorContext = {
  type: "calculator",

    userId: z.string().optional(),

      sessionStart: new Date().toISOString(),
      difficulty: args.difficulty,
    };
```

## Troubleshooting

### Common Issues

1. **Model Not Responding**: Check your API key and rate limits
2. **Memory Issues**: Reduce `maxEpisodes` or enable compression
3. **Slow Responses**: Adjust `maxTokens` or use a faster model
4. **TypeScript Errors**: Ensure all types are properly imported

### Debug Mode

Enable debug logging:

```typescript
const agent = createAgent({
  // ... other config
  debug: true,
  logLevel: "debug",
});
```

This will show detailed information about the agent's internal operations.
