## Getting Started with Examples

### Prerequisites

Before running any example, ensure you have:

1. **Node.js** (v18 or higher)
2. **pnpm** package manager
3. **API Keys** for your chosen model provider
4. **Environment Setup** completed

### Quick Setup

```bash
# Clone the repository
git clone https://github.com/your-org/axiomkit.git
cd axiomkit

# Install dependencies
pnpm install

# Set up environment variables
cp .env.example .env
# Edit .env with your API keys

# Run an example
cd app/examples/bots
pnpm run calculator-bot
```

## Example Customization

### Modifying Examples

Most examples can be easily customized:

```typescript
// Example: Customizing the Calculator Bot
import { createAgent } from "@axiomkit/core";
import { cliExtension } from "@axiomkit/cli";
import { groq } from "@ai-sdk/groq";

const customCalculatorExtension = createCliExtension({
  name: "advanced-calculator",
  instructions: [
    "You are an advanced mathematical calculator.",
    "Support complex operations: calculus, statistics, algebra.",
    "Show step-by-step solutions for educational purposes.",
    "Handle scientific notation and units.",
  ],
});

const agent = createAgent({
  model: groq("gemma2-9b-it"),
  modelSettings: {
    maxTokens: 1000,
    temperature: 0.1, // More precise for calculations
  },
  extensions: [customCalculatorExtension],
});
```

### Adding Custom Actions

```typescript
import { action } from "@axiomkit/core";
import { z } from "zod";

const customAction = action({
  name: "custom-operation",
  description: "Perform a custom operation",
  schema: z.object({
    input: z.string(),
    operation: z.enum(["process", "analyze", "transform"]),
  }),
  handler: async (args, ctx, agent) => {
    const { input, operation } = args;
    
    // Your custom logic here
    const result = await processInput(input, operation);
    
    return {
      success: true,
      result,
      timestamp: new Date().toISOString(),
    };
  },
});
```

## Example Best Practices

### 1. **Environment Management**
```bash
# Use environment-specific configurations
NODE_ENV=development
MODEL_PROVIDER=groq
API_KEY=your_api_key
MEMORY_PROVIDER=in-memory
```

### 2. **Error Handling**
```typescript
// Robust error handling in examples
const robustAgent = createAgent({
  model: groq("gemma2-9b-it"),
  extensions: [cliExtension],
  errorHandling: {
    retryAttempts: 3,
    logErrors: true,
    gracefulDegradation: true,
  },
});
```

### 3. **Testing Examples**
```typescript
// Test example functionality
describe("Calculator Bot", () => {
  it("should perform basic calculations", async () => {
    const result = await agent.run({
      context: calculatorContext,
      args: { operation: "add" },
      input: "2 + 2",
    });
    
    expect(result).toContain("4");
  });
});
```
