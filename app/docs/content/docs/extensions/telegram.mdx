# @axiomkit/telegram

Telegram bot integration for creating AI-powered Telegram bots with message processing and command handling.

## Installation

```bash
pnpm add @axiomkit/telegram telegraf
```

## Quick Start

```typescript
import { createAgent } from "@axiomkit/core";
import { telegramExtension } from "@axiomkit/telegram";
import { groq } from "@ai-sdk/groq";

const agent = createAgent({
  model: groq("gemma2-9b-it"),
  extensions: [telegramExtension],
  config: {
    telegram: {
      token: process.env.TELEGRAM_TOKEN,
    },
  },
});

await agent.start();
```

## Configuration

```typescript
interface TelegramConfig {
  token: string;           // Telegram bot token
  webhook?: {              // Webhook configuration
    url: string;
    port: number;
  };
  commands?: Command[];    // Custom commands
  middleware?: any[];      // Telegraf middleware
}
```

## Examples

### Custom Commands

```typescript
import { createTelegramExtension } from "@axiomkit/telegram";

const customExtension = createTelegramExtension({
  name: "ai-assistant",
  commands: [
    {
      command: "start",
      description: "Start the bot",
    },
    {
      command: "help",
      description: "Get help",
    },
    {
      command: "ask",
      description: "Ask a question",
    },
  ],
});

const agent = createAgent({
  model: groq("gemma2-9b-it"),
  extensions: [customExtension],
  config: {
    telegram: {
      token: process.env.TELEGRAM_TOKEN,
    },
  },
});
```

### Webhook Setup

```typescript
const agent = createAgent({
  model: groq("gemma2-9b-it"),
  extensions: [telegramExtension],
  config: {
    telegram: {
      token: process.env.TELEGRAM_TOKEN,
      webhook: {
        url: "https://your-domain.com/webhook",
        port: 3000,
      },
    },
  },
});
```
